# Push to GitHub

The repo is initialized and committed locally. To push:

1. **Create a new repo on GitHub** (don't add README or .gitignore)
   - Go to https://github.com/new
   - Name: `ai-governance-copilot`
   - Visibility: Public or Private

2. **Add remote and push:**
   ```bash
   cd ai-governance-copilot
   git remote add origin https://github.com/YOUR_USERNAME/ai-governance-copilot.git
   git push -u origin main
   ```

Replace `YOUR_USERNAME` with your GitHub username.
